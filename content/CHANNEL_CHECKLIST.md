# Master Channel Distribution Checklist

Use this list to ensure 100% coverage for every piece of content we produce.

---

## 1. Technical & Developer Channels
*Best for: "Show HN" pitch, Developer Cheat Sheets, WASM technical posts.*

- [ ] **Hacker News (Show HN):** Tuesday/Wednesday morning.
- [ ] **Dev.to:** Cross-post technical guides and manifestos.
- [ ] **Hashnode:** Great for SEO-focused technical articles.
- [ ] **GitHub:** Create a repository for the project (even if just for the README/Roadmap) and submit to "Awesome" lists (e.g., `awesome-self-hosted`, `awesome-web-utilities`).
- [ ] **Indie Hackers:** Share the "Product" and "Milestones" to the community.
- [ ] **Lobste.rs:** (If you have an invite) Highly technical audience similar to HN.

## 2. Utility & Startup Directories (The "Backlink Engine")
*Best for: Tool descriptions, "Alternative To" copy.*

- [ ] **AlternativeTo.net:** Priority #1. List as an alternative to ConvertCase.net.
- [ ] **Product Hunt:** Schedule a "Privacy-First" launch.
- [ ] **TinyHelpers.dev:** Submit for "Text Utilities."
- [ ] **StartupStash:** Submit under "Productivity" or "Development."
- [ ] **G2 / Capterra:** (If we position as "Software") Good for enterprise credibility.
- [ ] **SaaSHub:** Automatic discovery for software alternatives.

## 3. Community "Problem-Solving" (SSH Framework)
*Best for: Helpful templates, direct links to specific tools.*

- [ ] **Reddit:** 
    - `r/excel`, `r/writing`, `r/copyediting`, `r/webdev`, `r/privacy`, `r/productivity`.
- [ ] **Quora:** Answer questions like "What is a safe online case converter?".
- [ ] **Stack Overflow:** Add links to relevant answers *only* if they provide a GUI solution to a complex regex problem.
- [ ] **X/Twitter:** Use the "Search" function for keywords like "all caps help" or "formatting nightmare".

## 4. Professional & Writing Channels
*Best for: Manifesto, Excel Power-User Guide, Productivity tips.*

- [ ] **Medium:** The "Manifesto" article.
- [ ] **Substack:** Start a small newsletter for "Privacy-First Productivity."
- [ ] **LinkedIn:** Share the "Data Cleaning" guide for office professionals.
- [ ] **Niche Forums:**
    - **ProZ / TranslatorsCafe:** For translators dealing with casing issues.
    - **Legal Technology forums:** For lawyers cleaning up documents.

## 5. Privacy-First Communities
*Best for: The "Anti-Cloud" manifesto.*

- [ ] **Mastodon / Bluesky:** High concentration of privacy-conscious users.
- [ ] **PrivacyGuides.org (Forum):** Suggest it as a recommended local-only tool.
- [ ] **Lemmy / Kbin:** The decentralized alternatives to Reddit.
